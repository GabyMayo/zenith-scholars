# Zenith Scholars

[![React](https://img.shields.io/badge/React-18.2.0-61DAFB?style=flat&logo=react)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-5.4.21-646CFF?style=flat&logo=vite)](https://vitejs.dev/)

A professional, modern tutoring website for secondary school and International Baccalaureate (IB) mathematics and physics education. Built with React and Vite for optimal performance and developer experience.

## 🌟 Overview

Zenith Scholars provides comprehensive online tutoring services for students in secondary education and the International Baccalaureate program. Our platform offers personalized mathematics and physics instruction through experienced tutors with proven academic backgrounds.

### Key Features

- **📚 Comprehensive Course Coverage**: Secondary and IB-level mathematics and physics curricula
- **👨‍🏫 Expert Tutors**: Qualified instructors from top universities including Purdue, UPC, and University of Winnipeg
- **📱 Fully Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **⚡ Modern Tech Stack**: Built with React 18 and Vite for fast, reliable performance
- **🎨 Professional UI**: Clean, modern design with a sophisticated blue/navy color palette
- **📞 Multiple Contact Channels**: WhatsApp, Instagram, email, and payment integration

## 🚀 Quick Start

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone or download** the project to your local machine
2. **Navigate** to the project directory:
   ```bash
   cd zenith
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
5. **Open your browser** and navigate to `http://localhost:5173`

## 🛠️ Development

### Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint for code quality

### Customization

The website is designed to be easily customizable:

1. **Content Updates**: Modify course topics, tutor information, and pricing in `src/App.jsx`
2. **Styling**: Adjust colors, fonts, and layout in `src/styles.css`
3. **Components**: Extend or modify individual components in `src/components/`
4. **Contact Information**: Update social media links and contact details

## 📱 Responsive Design

The website features a fully responsive design that adapts seamlessly across all device sizes:

- **Desktop** (>768px): Multi-column layouts with full feature set
- **Tablet** (768px-980px): Adapted grid layouts and spacing
- **Mobile** (<768px): Single-column stacking with optimized touch targets

## 🎨 Design System

- **Color Palette**: Professional blue/navy theme with soft gradients
- **Typography**: Inter font family for modern, readable text
- **Components**: Rounded cards with subtle shadows and hover effects
- **Spacing**: Consistent 8px grid system with responsive breakpoints
