# Zenith Scholars

A modern tutoring website built with React and Vite for secondary school and IB mathematics and physics.

## What is included

- Home page with hero section, course previews, tutor previews, pricing preview, and contact details
- Courses section for Secondary and IB learning paths
- Tutors section with profile cards and academic details
- Pricing section with clean responsive cards
- Contact section with WhatsApp, Instagram, Gmail, and Yape/Plin placeholders
- Modern responsive styling with a soft blue/navy palette and rounded cards

## Project structure

- `index.html` — main HTML file
- `package.json` — project dependencies and scripts
- `vite.config.js` — Vite build config
- `src/main.jsx` — app entry point
- `src/App.jsx` — main component structure
- `src/styles.css` — global styles and layout
- `src/components/` — reusable UI components

## Setup

1. Open a terminal in `c:\Users\Lola\Downloads\zenith`
2. Run `npm install`
3. Run `npm run dev`
4. Open the local URL shown by Vite in your browser

## Notes for beginners

- `npm install` downloads the packages React and Vite
- `npm run dev` starts the development server so you can preview the website
- Edit files in `src/` to change the page content and styles
- The app is mobile responsive, so it works across phones and desktop screens
